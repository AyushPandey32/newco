# VentureLink - Student-Led Micro-VC Platform

A professional, modern landing website for a student-led micro-VC platform that sources and supports early-stage startups in undercapitalized regions (Asia, Africa, Latin America).

## Features

### ✨ Complete Landing Page Sections
- **Hero Section**: Compelling value proposition with animated graphics
- **About Us**: Mission statement, student network highlights, and key statistics
- **For Startups**: Benefits and application form (ready for Typeform/Airtable integration)
- **For Investors**: Partnership opportunities and collaboration details
- **Blog Section**: Thought leadership content showcase
- **Footer**: Complete site navigation and social links

### 🎨 Modern Design
- Clean, professional aesthetic
- Gradient color scheme with purple/blue theme
- Smooth animations and hover effects
- Interactive elements and micro-interactions
- Responsive design for all devices

### 📱 Mobile-Friendly
- Fully responsive layout
- Mobile navigation menu
- Touch-friendly interactions
- Optimized for all screen sizes

### ⚡ Performance Features
- Smooth scrolling navigation
- Intersection Observer animations
- Optimized loading animations
- Progressive enhancement

## Quick Start

1. **Open the website**: Simply open `index.html` in your web browser
2. **Customize content**: Edit the HTML file to update text, images, and links
3. **Add your form**: Replace the form placeholder with your actual Typeform or Airtable embed
4. **Deploy**: Upload files to your web hosting service

## Customization Guide

### Adding Your Application Form

Replace the form placeholder in the "For Startups" section:

```html
<!-- Replace this placeholder -->
<div class="form-placeholder">
    <!-- ... placeholder content ... -->
</div>

<!-- With your actual form embed -->
<iframe src="https://form.typeform.com/to/YOUR_FORM_ID" width="100%" height="500px"></iframe>
```

### Updating Content

- **Company Name**: Search and replace "VentureLink" with your platform name
- **Statistics**: Update the numbers in the About section (50+ Student Partners, etc.)
- **Contact Email**: Replace `partners@venturelink.com` with your actual email
- **Social Links**: Update the footer social media links
- **Blog Posts**: Replace sample blog posts with your actual content

### Color Scheme

The website uses CSS custom properties for easy color customization. Main colors:
- Primary: `#667eea` (Purple-blue gradient start)
- Secondary: `#764ba2` (Purple gradient end)
- Background: Various light gradients

### Adding Real Images

Replace the placeholder image elements with actual images:
- Hero section graphic
- Blog post images
- Team photos (if desired)
